## Cara Menggunakan

   ```bash

   ```

2. Buka file-file PHP dengan editor teks favorit Anda

## Prasyarat

Untuk mengikuti materi dalam repositori ini, Anda membutuhkan:

- [PHP 7.4 atau lebih tinggi](https://www.php.net/downloads)
- Server web lokal (XAMPP, Laragon, WAMP, atau MAMP)
- Editor teks (VS Code, Sublime Text, Atom, dll)
- Pengetahuan dasar HTML dan CSS (akan membantu)
